## 📊 프로젝트 개요
- **Bird-eye View 변환** (차선 영역을 더 잘 보이도록 투영 변환 적용)
- **Sobel Edge Detection 활용** (차선 경계를 강조)
- **Color Filtering & Masking 적용** (노란색, 흰색 차선을 강조)
- **조명 변화 보정 (CLAHE 적용)** (어두운 환경에서도 차선 검출 가능)
- **차선 검출 실패 시 예외 처리** (안정적인 검출 성능 유지)
- **실시간 차선 인식 & 결과 영상 저장**
